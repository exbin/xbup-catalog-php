
                            XBUP Catalog (XBCatalog)
                            ========================

This is experimental implementation of XBUP Catalog written in PHP.

Homepage: http://xbup.exbin.org
License: GNU/LGPL (see lgpl-3.0.txt)
Requires: PHP http://www.php.net/
          MariaDB http://mariadb.com/
          Apache Web server (or similar) http://httpd.apache.org/

Settings:

You need to edit db.php and interface/db.php files relevantly to your configuration.

You need to import catalog.sql file which creates account:
login: admin
passwd: admin
